# SECRETE A Personal Private Journal
### Technologies used in this project:
- HTML
- CSS
- Javascript
- PHP
- MYSQL

## Get started
- Install XAMPP in your machine
- Make a database called 'secretedb'
- Import the tables from the DATABASE folder
